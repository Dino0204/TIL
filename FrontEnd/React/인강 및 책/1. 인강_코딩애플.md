## 2. 스테이트

- State 사용법:
  1. import {useState}
  2. useState(보관할 자료)
  3. let[작명,작명]
- State를 사용해야 하는 경우:
  → 변수는 값 변경 시 즉각 반영이 x
  → State는 자동으로 Html에 즉각 반영
- State 변경법:
  - 등호로 변경금지
  - State 변경 함수 사용
  - 배열 값 변경 시 :
    - cpy 사용 (원래 값 필요할 수 있으니)
    - […logo] 같이 나머지를 복사하는 문법 사용
  - State 변경 시 원래 State 값과 같다면 변경 x
